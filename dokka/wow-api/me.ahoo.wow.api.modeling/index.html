interface NamedAggregateDecorator : NamedAggregate

NamedAggregateDecorator接口定义了一个装饰器模式的命名聚合根。 它继承自NamedAggregate，并委托实际的命名聚合根实现。 这个接口允许在不修改原有聚合根逻辑的情况下，动态添加功能。
