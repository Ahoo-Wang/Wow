class PaginationDsl

A DSL for constructing Pagination objects.

This class provides a fluent API to configure and build Pagination instances, allowing the user to specify the index and size of the pagination. The index represents the current page number, and the size represents the number of items per page.

Example usage:

val pagination = pagination {
    index(2)
    size(15)
}
